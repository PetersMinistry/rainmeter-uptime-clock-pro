# Uptime Clock Pro Font Credits

These fonts are included for development and distribution planning. They are licensed under the SIL Open Font License, included beside the font files.

## Font Stack

- WindSong by the Google Fonts project: signature clock face.
- Manrope by the Google Fonts project: readable UI text, values, and controls.
- Outfit by the Google Fonts project: smooth modern headings and panel labels.
- Cormorant Garamond by the Google Fonts project: bundled as an alternate refined display face.

## Source URLs

- https://fonts.google.com/specimen/WindSong
- https://fonts.google.com/specimen/Manrope
- https://fonts.google.com/specimen/Outfit
- https://fonts.google.com/specimen/Cormorant+Garamond

## Install Notes

Rainmeter uses installed Windows font families. Install the `.ttf` files in this folder, then refresh Rainmeter or restart it if the new font names do not appear immediately.
